# 🎓 Missão Conecta EDN - Exercícios Python

Repositório com exercícios práticos realizados durante a Missão Conecta EDN pela Escola da Nuvem.

📚 Exercícios

### 1️⃣ Cadastro Interativo (`exercicio_1_cadastro.py`)
Programa que faz cadastro interativo de alunos com validação de matrícula.

**Funcionalidades:**
- Entrada de nome, idade e status de matrícula
- Validação de entrada do usuário
- Exibição formatada dos dados

---

2️⃣ Ano de Nascimento (`exercicio_2_ano_nascimento.py`)
Calcula a idade de uma pessoa baseado no ano de nascimento.

**Funcionalidades:**
- Entrada do nome e ano de nascimento
- Cálculo automático da idade
- Exibição personalizada

---

3️⃣ Calculadora com Input (`exercicio_3_calculadora.py`)
Calculadora que realiza operações básicas com dois números.

**Funcionalidades:**
- Soma
- Subtração
- Multiplicação
- Divisão (com validação para divisão por zero)

---

4️⃣ Número Par ou Ímpar (`exercicio_4_par_impar.py`)
Verifica se um número é par ou ímpar.

**Funcionalidades:**
- Entrada de um número inteiro
- Verificação usando operador módulo (%)
- Resultado formatado

---

5️⃣ Divisão Inteira e Resto (`exercicio_5_divisao.py`)
Calcula a divisão inteira e o resto de uma divisão.

**Funcionalidades:**
- Entrada de dois números
- Cálculo da divisão inteira (//)
- Cálculo do resto (%)
- Validação para divisão por zero
